# 《和平精英》AI队友行为数据标注分析示例

这是一个为申请AI训练（数据标注）实习岗位而创建的模拟项目。项目展示了如何将《和平精英》中的复杂游戏行为解构为可供AI模型学习的结构化数据标签。

## 项目目的
*   理解AI数据标注工作的核心流程。
*   将玩家抽象的“游戏意识”转化为具体的、可标注的行为标签。
*   展示对数据标注工作所需严谨性和洞察力的理解。

## 核心结论
高质量的标注不仅是分类，更是对**行为意图**的理解。例如，“移动”在不同的战术场景下（如“利用枪声掩护转移” vs. “正面突击”）应具有不同的标签，这直接决定了未来AI队友的智能水平。
